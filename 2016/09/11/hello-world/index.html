静态微博安装

安装环境（只需安装一次）

npm install nrm -g (全局下安装)
nrm ls 源目录

nrm use tabao (改变源下载地址)

安装hexo

npm install hexo-cli -g（window）

sudo npm install hexo-cli -g（mac安装）

初始化blog

hexo init blog

写好的文件会在我们的source中

在本地进行查看

-当前服务不能断开

hexo server

##发布到github上
需要创建一个仓库名字必须为 username.github.io

提交到git上需要安装git插件

npm install hexo-deployer-git –save (必须在打开的文件中安装)

配置发布信息

在根目录下的_config.yml 文件

deploy:
type: git
要求的格式 repo: https://账号:密码@地址github.com/仓库名
repo: https://yangtongyier:yt172073@github.com/yangtongyier/yangtongyier.github.io.git
branch: master

message: push

##推送到github项目上

hexo deploy

##第二次发布之前要执行命令

hexo g
